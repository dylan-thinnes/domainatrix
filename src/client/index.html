Domainatrix requires JavaScript

Please enable JavaScript to use this site.

We understand the concerns you may have about JavaScript, but we need it for DOM manipulation, and the main interface currently uses ReactJS.

For the old non-React interface, click here

The Non-React interface's JavaScript:
- Is all in a single file <20kB in size.
- Is all self-written.
- Only makes requests to its own domain and server.
- Has no ads, no tracking, no sensitive data, no analytics.
- Has no dependencies.
